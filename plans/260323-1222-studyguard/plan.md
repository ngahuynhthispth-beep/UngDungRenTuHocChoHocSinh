# Plan: StudyGuard - App Giám Sát Học Sinh Tự Học
Created: 23/03/2026
Status: 🟡 In Progress

## Overview
Web App sử dụng camera + AI (MediaPipe) để giám sát học sinh tự học tại nhà. Nhận diện 3 trạng thái: Đang học / Mất tập trung / Không học. Phát cảnh báo giọng nói + gửi thông báo cho phụ huynh.

## Tech Stack
- Frontend: HTML + CSS + JavaScript (Vanilla)
- AI Engine: MediaPipe Pose Landmarker (chạy trên browser)
- Voice: Web Speech API
- Real-time: WebSocket (Socket.io)
- Backend: Node.js + Express
- Database: SQLite (better-sqlite3)
